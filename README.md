# Decision Lens Skill

A Codex skill for high-fidelity decision making under ambiguity.

It helps an agent:
- prioritize by value versus cost
- reduce decision entropy
- apply systems thinking, first principles, second-order thinking, inversion, and Bayesian updates
- produce clear recommendations instead of option dumps

## Repository Structure

```text
skills/
  decision-lens/
    SKILL.md
    references/
      decision-framework.md
```

## Install

Copy or symlink `skills/decision-lens` into your Codex skills directory, for example:

```bash
mkdir -p ~/.codex/skills
cp -R skills/decision-lens ~/.codex/skills/
```

## Trigger Examples

Use this skill for prompts like:
- "Help me think through this strategy choice"
- "Which option should we prioritize?"
- "Is this even worth building?"
- "Give me a first-principles take"
- "Analyze the tradeoffs and recommend a path"

## What the Skill Optimizes For

- value-first decisions
- lower complexity and lower ambiguity
- explicit tradeoffs
- anti-fragile plans
- clear next actions
